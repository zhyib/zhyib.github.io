GitHub Actions 与 Gitee 自动部署

Posted on 2021-09-24 Edited on 2024-05-04 In 技术 , 杂谈 Views:

简介

GitHub 的自动化流水线，主要功能为

  • 监听推送分支
  • 自动构建并部署支 GitHub Pages
  • 自动推送至 Gitee
  • 将构建完成的内容也推送至 Gitee Pages

./.github/workflow/pages.yml

只要创建了这个文件再去项目顶部标签的 Actions 配置即可

简单加点备注，留存个笔记

说明

公私钥

  1. 在本机上生成 ssh 密钥 ssh-keygen
  2. 给 Github项目设置配置私钥 id_rsa 因为是ssh链接发起方
  3. 在 Gitee 的个人设置页面「安全设置 -> SSH 公钥」 配置 SSH 公钥，因为是链接的接收方

GitHub

最终有两个配置

  • Gitee 密码
  • ssh 私钥
